# Contributing Guide

## Cách đóng góp cho dự án

### 1. Issues

- Báo cáo lỗi: Cung cấp thông tin chi tiết, bước tái tạo, và kết quả mong đợi
- Đề xuất tính năng: Mô tả use case và lợi ích

### 2. Pull Requests

- Fork repository
- Tạo branch mới: `git checkout -b feature/your-feature`
- Commit thay đổi: `git commit -m 'Add feature'`
- Push: `git push origin feature/your-feature`
- Mở PR

### 3. Code Style

- Tuân thủ PEP 8
- Sử dụng type hints
- Viết docstrings
- Thêm unit tests

### 4. Commit Messages

- `feat:` - Tính năng mới
- `fix:` - Sửa lỗi
- `docs:` - Tài liệu
- `test:` - Tests
- `refactor:` - Refactoring
