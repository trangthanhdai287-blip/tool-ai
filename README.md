# tool-ai
tool ai cho kali linux
## Hướng dẫn cài đặt nhanh
1. Tải tool về:
   `git clone https://github.com/trangthanhdai287-blip/tool-ai`
2. Cấp quyền và chạy:

   `cd tool-ai`
   `chmod +x setup.sh`
   `./setup.sh`
## Dùng lệnh 
1 Mở ai:

 `ai`
 
2 Khi có bản mới dùng lệnh này:

 `Hủy các thay đổi cục bộ của tệp bị lỗi`
 `git checkout setup.sh`

`Sau đó thực hiện lại lệnh cập nhật`
`git pull origin main`
